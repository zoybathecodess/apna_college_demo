Git Command Cheat Sheet

Concise list of common Git commands grouped by category. Copy & paste the commands as needed.

Basic setup

CommandDescription
git config --global user.name "your_name"Set global username
git config --global user.email "your_email"Set global email
git config --listShow current git config

Repository setup

CommandDescription
git initInitialize a new Git repository
git clone "url"Clone remote repository
git remote add origin "url"Add remote repository
git remote -vShow remotes and their URLs

File management

CommandDescription
git statusShow working tree status
git add .Add all changes to staging area
git add filenameAdd a specific file to staging
git commit -m "message"Create a commit with message
git commit -am "message"Add & commit tracked files (shortcut)
git rm "file_name"Remove file from working dir & staging
git rm --cached "file_name"Remove file from staging only (keep locally)
git mv "old" "new"Rename or move a file

Branching & merging

CommandDescription
git branchList branches
git branch branch_nameCreate a new branch
git checkout branch_nameSwitch to a branch
git checkout -b branch_nameCreate & switch to a new branch
git merge branch_nameMerge branch into current
git rebase branch_nameReapply commits on top of another branch
git cherry-pick commit_idApply a specific commit from another branch

Undo & reset

CommandDescription
git reset file_nameUnstage a file (keeps local edits)
git reset --hard commit_idReset working tree & index to a specific commit (destructive)
git checkout -- file_nameDiscard local changes in a file
git revert commit_idUndo a commit by creating a new commit
git clean -fdRemove untracked files and directories

Stash

CommandDescription
git stashSave local changes temporarily
git stash listList stashed changes
git stash popApply last stash and remove it from the stash list
git stash apply stash@{n}Apply a specific stash (keep it in list)
git stash drop stash@{n}Delete a specific stash

Remote operations

CommandDescription
git push -u origin mainPush to remote and set upstream
git pushPush commits to remote
git push origin branch_namePush a specific branch
git pull origin mainFetch & merge from remote branch
git fetchFetch remote changes without merging

Logs & history

CommandDescription
git logShow commit history
git log --onelineCompact history (one commit per line)
git log --graph --oneline --allVisualize branches and commits
git diffShow unstaged changes
git diff --stagedShow staged changes
git blame file_nameShow who changed each line
git shortlogSummary of commits by author

Tags & releases

CommandDescription
git tagList tags
git tag tag_nameCreate lightweight tag
git tag -a v1.0 -m "message"Create annotated tag
git show tag_nameShow tag details
git push origin tag_namePush a tag
git push origin --tagsPush all tags

Archiving & export

CommandDescription
git archive -o archive.zip HEADCreate a zip archive of the current branch
git bundle create repo.bundle --allCreate a portable repository bundle

Advanced

CommandDescription
git reflogView reference log of HEAD (all movements)
git bisect startStart binary search to find a bad commit
git bisect good commit_idMark a commit as good during bisect
git bisect bad commit_idMark a commit as bad during bisect
git bisect resetEnd bisect and return to normal
git submodule add url pathAdd a submodule
git submodule update --init --recursiveInit & update submodules recursively
